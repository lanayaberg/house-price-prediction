# House Price Prediction

ML-проект для предсказания цены дома на основе его характеристик. Полный pipeline: от очистки данных до работающего веб-приложения.

## Демо

[Ссылка на работающее приложение](сюда_вставишь_ссылку_после_деплоя)

## Задача

Предсказать цену дома (`SalePrice`) на основе его характеристик: площадь, год постройки, качество отделки и др. Датасет: [House Prices - Kaggle](https://www.kaggle.com/c/house-prices-advanced-regression-techniques)

## Использованные инструменты

- pandas, numpy — обработка данных
- matplotlib, seaborn — визуализация
- scikit-learn, xgboost — модели машинного обучения
- streamlit — веб-интерфейс
- joblib — сохранение модели

## Процесс работы

1. **EDA и очистка данных** — обработка 19 колонок с пропусками, различие между "пропуск = нет данных" и "пропуск = нет такой характеристики" (например, бассейна)
2. **Логарифмирование целевой переменной** — устранение скошенности распределения цен (skew снизился с 1.88 до 0.12)
3. **One-Hot Encoding** — превращение 43 категориальных признаков в числовой формат (262 итоговых колонки)
4. **Сравнение моделей**:
   - Linear Regression — R² 0.88
   - Random Forest — R² 0.88
   - XGBoost — R² 0.89 (лучшая модель)
5. **Веб-интерфейс на Streamlit** — пользователь вводит параметры дома и получает предсказанную цену

## Как запустить локально

\`\`\`bash
pip install -r requirements.txt
python eda.py
streamlit run app.py
\`\`\`
